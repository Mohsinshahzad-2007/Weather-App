# Weather App

A responsive web application that provides real-time weather forecasts and 3-day high/low predictions. Built for learning web development and API integration.

Features
* **Live Weather Data**: Get current temperature and conditions for any city.
* **3-Day Forecast**: View high and low temperatures for the upcoming days.
* **Responsive Design**: Looks great on both desktop and mobile screens.

Tech Stack
* **Framework**: React.js with Vite
* **Styling**: CSS3
* **API**: [Insert API Name, e.g., OpenWeatherMap]

How to Run Locally
1. Clone this repository:
   `git clone https://github.com/Mohsinshahzad-2007/Weather-App.git`
2. Navigate to the project folder:
   `cd Weather-App`
3. Install dependencies:
   `npm install`
4. Start the development server:
   `npm run dev`
